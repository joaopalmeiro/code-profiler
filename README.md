# code-profiler

A Node.js CLI to generate a [Profile file](https://code.visualstudio.com/updates/v1_75#_profiles) for VS Code from a `.vscode` folder.

## Development

```bash
nvm install && nvm use && node --version
```

```bash
npm install
```

```bash
npm run dev -- --help
```

```bash
npm run dev -- Node
```
